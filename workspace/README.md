# Enhanced Abstract Animation with Anime.js

An elegant, interactive abstract animation featuring multiple shapes with fluid motion, color transformations, and subtle user interactions.

## Features

- **Multiple Animated Shapes**: 5 distinct shapes with individual animations
- **Fluid Motion**: Orbital paths, scaling, rotation, and border radius transformations
- **Color Transitions**: Dynamic color changes from sophisticated palettes
- **Interactive Elements**: 
  - Hover effects on shapes with elastic scaling
  - Mouse-follow subtle interactions
  - Responsive design
- **Visual Enhancements**:
  - Glassmorphism container effect
  - Gradient backgrounds with radial effects
  - Subtle glows and shadows
  - Animated floating particles
- **Layered Animations**: Multiple timelines for complex, organic movement

## Technologies Used

- HTML5
- CSS3 (with modern features like glassmorphism, gradients, filters)
- [Anime.js](https://animejs.com/) (JavaScript animation library)

## How to Use

Simply open `index.html` in any modern web browser to view the animation.

## Customization

You can modify the following aspects:

### In `style.css`:
- Adjust container size, colors, and effects
- Modify shape initial positions and sizes
- Change background gradients
- Tweak glassmorphism and particle effects

### In `script.js`:
- Adjust animation durations, delays, and easing functions
- Modify color palettes
- Change interaction sensitivities
- Add or remove shapes (remember to update HTML accordingly)

## Development

This project demonstrates advanced CSS and JavaScript animation techniques using Anime.js for creating generative art-like animations in the browser.

## Browser Support

Works in all modern browsers that support:
- CSS3 transforms, transitions, and filters
- ES6 JavaScript
- Anime.js library
